# Bills. Bills everywhere.

## Start

`npm install` to install the packages

`npm run api` to run the server

`npm start` to start the app

`npm run test` to run the tests

## Built with

- typescript
- react
- react-redux
- redux-saga
- styled-components
- react-testing-library
